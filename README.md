# webpack-ant-design-demo
####webpack-ant-design-demo webpack ant design的样例
####这是一个不同于官网给出的案例的webpack+ant的样例
    1.利用了webpack-dev-server方便本地的开发
    2.可用来本地作为基础项目做开发
    3.用了happypack做上线的多线程开发
    4.用了watch模式，监听文件的修改

运行方式:

###1:安装依赖:
    2npm install
###2:启动项目
    2npm start
